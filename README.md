# Odin Landing Page

A responsive landing page project built as part of The Odin Project Foundations course.  
This project focuses on improving HTML and CSS skills by creating a clean and modern landing page layout using Flexbox.

---

## 🚀 Live Demo
https://mananbdev.github.io/Odin-Landing-page/

---

## ✨ Features

- Responsive landing page design
- Modern UI layout
- Flexbox-based sections
- Hero section
- Information cards
- Quote/Testimonial section
- Call-to-action section
- Footer section

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox

---

## 📚 What I Learned

Through this project, I practiced:

- Structuring webpages properly
- Using Flexbox for layouts
- Responsive design principles
- Spacing and alignment
- Typography and section design
- Writing cleaner HTML and CSS
